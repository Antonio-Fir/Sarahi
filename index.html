JavaScript es un lenguaje de programación o de secuencias de comandos que te permite implementar funciones complejas en páginas web

Es la tercera capa del pastel de las tecnologías web estándar, dos de las cuales son (HTML y CSS)

Cuando cargas una página web en tu navegador, estás ejecutando tu código (HTML, CSS y JavaScript) dentro de un entorno de ejecución

La asincronía es uno de los pilares fundamentales de Javascript. Es profundizar en las piezas y elementos que la hacen posible.
